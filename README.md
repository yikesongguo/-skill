# 🧳 Summer Trip — AI 旅行规划助手

> QoderWork Skill：通过自然对话了解旅行需求，联网搜索真实数据，生成排版精美的 Word 行程手册。

## 功能特性

- **自然对话引导**：像旅行顾问一样聊天，逐步收集出行类型、人数、目的地、天数、预算、体力状况等信息
- **联网真实数据**：景点门票、火车时刻、餐厅价格、天气预报均来自实时搜索（12306、大众点评、高德地图等），不编造
- **多城市行程**：支持跨城市旅行规划，自动分配天数和交通衔接
- **智能推荐**：根据用户预算、兴趣偏好（自然/人文/美食/冒险/休闲）和体力状况筛选景点和餐厅
- **网友真实评价**：每个景点和美食附带真实好评和差评，帮助用户做决策
- **优惠身份识别**：学生证、老人、军人、儿童等证件自动计算门票优惠
- **迭代调整**：每轮推荐后收集反馈，反复调整直到满意
- **Word 文档输出**：生成可编辑的 `.docx` 行程手册，含封面、每日行程、预算明细、实用贴士

## 服务范围

| 支持 | 不支持 |
|------|--------|
| 中国大陆所有城市 | 出境游（推荐携程/飞猪国际版） |
| 港澳台（签注/入台证提醒） | 极端活动（5000m+登山/深潜/漂流） |
| 团队 ≤ 10 人 | 医疗建议、代预订服务 |
| 自驾（推荐景点/美食/住宿，不规划路线） | 违规操作（逃票/野景点） |

## 安装与使用

### 1. 安装依赖

```bash
pip install python-docx
```

### 2. 安装为 QoderWork Skill

将 `summer-trip` 目录复制到 QoderWork 技能目录：

- **Windows**: `C:\Users\<用户名>\.qoderworkcn\skills\summer-trip\`
- **macOS**: `~/.qoderworkcn/skills/summer-trip/`

或在 QoderWork 中通过技能管理界面导入。

### 3. 触发方式

在 QoderWork 对话中提到以下关键词即可触发：

> 旅行计划、旅游攻略、行程安排、去哪玩、旅行、度假、自由行、假期出游、周末游、暑假出行、港澳游、台湾游...

### 4. 对话流程

1. **开场介绍**：AI 说明功能和使用方式
2. **需求收集**：通过选择题和自然对话了解出行信息
3. **信息确认**：展示收集到的信息摘要，用户确认
4. **搜索推荐**：联网搜索景点、美食、交通、住宿
5. **行程草案**：组合生成按天排列的行程草案
6. **迭代调整**：用户反馈，AI 调整，反复确认
7. **生成文档**：用户满意后生成 Word 行程手册
8. **交付后续**：提供预订提醒和后续调整支持

## 技术细节

### JSON 数据结构

行程数据通过 JSON 传递给 `build_trip_docx.py`，主要字段：

- `travelers`: 出行人信息（人数、类型、优惠身份）
- `days[]`: 每日行程（日期、主题、住宿、活动列表）
- `activities[]`: 活动详情（标题、分类、时长、费用、交通、景区内路线）
- `budget`: 预算明细（交通/住宿/门票/餐饮/其他 + 总计）
- `weather`: 天气摘要（温度、降雨、穿衣建议、风险提示）
- `tips`: 实用贴士（行李清单、注意事项、紧急联系）

完整 schema 见 `references/docx-layout.md`。

### 数据校验

生成文档前会自动校验：

- 题号连续性和答案范围
- 预算数学一致性（各分类 total 之和 = grand_total）
- packing 数量范围（9-15 项）
- transit 覆盖（非首活动必须有 ≥2 种交通方式）
- category 枚举值合法性

### Word 文档规格

- **纸张**：A4，上下 2cm / 左右 2.5cm 页边距
- **字体**：微软雅黑，正文小四（12pt），标题四号（14pt）
- **配色**：深蓝标题 #1B5E7B，海洋蓝强调 #2E86AB，红色费用 #C73E1D
- **分类颜色**：交通蓝、景点橙、餐饮红、住宿绿、购物紫、活动深橙
