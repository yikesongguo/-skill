# 美食搜索与推荐指南

本文档定义美食搜索策略、展示格式和筛选规则。执行第 4 步（美食推荐）前阅读。

## 搜索维度

美食搜索分**两个维度**，都要搜：

### 维度一：景点附近

用户选的景点附近有什么好吃的，方便就近用餐：

```
搜索关键词：
1. "{城市} {景点名} 附近 高分餐厅 大众点评"
2. "{城市} {景点名} 附近 美食推荐"
3. "{城市} {景点名} 附近 好吃的小店"
4. "{城市} {景点名} 附近 美食 知乎"
```

每个景点附近至少搜 4-5 家。

### 维度二：城市热门

不限于景点位置，搜该城市整体值得去的餐厅、美食街、老字号：

```
搜索关键词：
1. "{城市} 必吃美食 本地人推荐"
2. "{城市} 美食街 推荐"
3. "{城市} 老字号 特色餐厅"
4. "{城市} 大众点评 必吃榜"
5. "{城市} 美食攻略 知乎"
6. "{城市} 必吃餐厅 马蜂窝"
7. "{城市} 美食林 携程"
```

### 维度三：特色早餐

很多城市有独特的早餐文化，必须单独搜索，让每天的行程都有早餐推荐：

```
搜索关键词：
1. "{城市} 特色早餐 本地人推荐"
2. "{城市} 早餐 必吃"
3. "{城市} 早点 老字号"
```

早餐安排原则：
- 结合当天第一个景点的位置，推荐顺路或附近的早餐店
- 早餐店通常人均较低（¥8-20），不受用户午餐/晚餐价位偏好限制
- 有特色早餐文化的城市（如洛阳汤馆、广州早茶、武汉热干面、西安肉丸胡辣汤）必须重点推荐
- 如果用户提到想吃夜宵，额外搜索 `"{城市} 夜宵 深夜美食 推荐"`

## 特色美食标注

- 特色美食必须标注 **[特色]** 标签，让用户一眼区分
- 普通好吃的店铺**不标注**
- 判断标准：是否为该城市/地区的传统特色、非遗美食、本地人公认的代表性食物

## 价位筛选

- 人均消费必须在用户选的价位区间内
- 超出区间的不推荐
- 用户选了"不限"时，按中等水平推荐

## 展示格式

按两组分区展示：

```
根据你的价位（每餐 ¥XX-XX），推荐这些好吃的：

景点附近：
{景点名}附近：
  1. 餐厅名 — 菜系 | 人均 ¥XX | 评分 X.X
     推荐: 菜1、菜2、菜3
  2. [特色] 特色店名 — 特色小吃 | 人均 ¥XX | 评分 X.X
     推荐: 特色菜1、特色菜2
  3. 小店名 — 类型 | 人均 ¥XX | 评分 X.X
     推荐: 菜1、菜2

城市热门 / 美食街：
  1. 美食街名 — 推荐: 店1、店2、店3（步行可逛多家）
  2. 餐厅名 — [特色] 菜系 | 人均 ¥XX | 评分 X.X
     推荐: 菜1、菜2（距XX景点X分钟车程，值得专程去）
  3. 老字号名 — 类型 | 人均 ¥XX | 评分 X.X
     推荐: 招牌菜1、招牌菜2

标注 [特色] 的是当地特色美食，其他是评分高的好吃店铺。
有想去的吗？或者有忌口/不想吃的？
```

## 每家店必须包含的信息

- 餐厅名称
- 菜系/类型
- 人均消费（含 ¥ 符号）
- 平台评分（大众点评/高德评分，直接写小数如 4.5；餐厅不用星级，星级仅用于景区）
- 推荐菜品（至少 2-3 道）
- [特色] 标签（仅特色美食标注）

## 信息来源与可信度分级

不同平台的广告污染程度不同，按可信度分三级：

**高可信（优先采信）**：
- **大众点评必吃榜/黑珍珠**：平台编辑筛选，有质量门槛
- **携程美食林**：携程编辑筛选的餐厅榜单
- **高德/百度地图评分**：基于大量用户真实导航和到店数据
- **马蜂窝美食攻略**：旅行社区，长文攻略比碎片种草更可信

**中可信（参考但需验证）**：
- **大众点评普通评分**：整体可靠但存在刷好评现象，重点看评分分布而非单一分数
- **知乎本地人回答**：长回答通常真实，但要警惕营销号
- **小红书美食攻略**：内容丰富但软植入多，需要交叉验证

**低可信（不单独作为推荐依据）**：
- **抖音/快手探店视频**：付费推广比例极高，不作为推荐来源
- **微博美食推荐**：营销号泛滥
- **任何平台的"广告"/"推广"/"合作"标记内容**：直接排除

## 交叉验证规则

**核心原则：一家店至少在 2 个独立来源出现才推荐。**

| 出现次数 | 处理方式 |
|----------|----------|
| 仅 1 个来源提到 | 不推荐，除非来源是高可信级且是必吃榜/美食林等编辑榜单 |
| 2 个独立来源提到 | 可以推荐 |
| 3 个及以上来源提到 | 优先推荐，可信度高 |

**"独立来源"的定义**：大众点评和美团不算独立来源（同一母公司）。小红书和知乎算独立来源。

## 软文/广告识别清单

搜索结果中出现以下任一信号，该店铺**不推荐**：

**内容信号**：
- 描述中大量使用"绝绝子""yyds""必打卡"等营销话术但无具体菜品描述
- 所有评价都是五星好评，没有任何中评或差评
- 多条评论用词高度雷同（疑似水军刷评）
- 内容只推一家店，不提及同区域其他选择（疑似定向推广）

**账号信号**：
- 推荐账号历史内容几乎全是不同店铺的"种草"（疑似职业探店号）
- 账号粉丝量高但评论互动极少（疑似买粉）

**商业信号**：
- 内容中出现"团购链接""优惠码""限时折扣"等促销信息
- 内容明确标注"广告""推广""品牌合作""商业推广"
- 店铺近期突然大量出现好评（疑似开业刷单期）

**遇到疑似软文时的处理**：
1. 标记该店铺为"疑似推广"，跳过
2. 从搜索结果中找下一家，优先选有多年稳定评价的老店
3. 如果搜到的结果中软文比例过高，换关键词重新搜索（如加"本地人""老字号"等限定词）

## 搜索失败恢复

- 景点附近搜不到足够餐厅 → 扩大搜索范围（如"附近 3 公里"）
- 城市热门搜不到结果 → 告知用户"这个城市的美食信息较少，到了可以看大众点评临时找"
- 搜到的餐厅全部超出用户价位 → 放宽搜索条件或建议用户调高预算
- 搜索结果中软文/广告比例过高 → 换关键词重搜，加"本地人推荐""老字号""苍蝇馆子"等限定词过滤广告内容
