# 景区门票优惠政策

本文档提供中国景区门票通用优惠政策参考。AI 收集用户的优惠身份后，根据此表在推荐景点时标注可能享受的优惠。

> 各地执行细则可能有差异，以下为通用规则。出行前建议查阅目标景区官网确认。

---

## 免票人群

| 人群 | 条件 | 有效证件 |
|---|---|---|
| 老年人 | 65 周岁以上（部分景区 70 周岁以上） | 身份证、老年证 |
| 儿童 | 6 周岁以下 或 身高 1.2m 以下 | 户口本、身份证 |
| 现役军人 | 含配偶和未成年子女 | 军官证、士兵证、士官证 |
| 退役军人 | 持优待证（部分景区全免，部分鼓励减免） | 退役军人优待证 |
| "三属"人员 | 烈士遗属、因公牺牲军人遗属、病故军人遗属 | 优待证 |
| 残疾军人 | 含军队离退休干部 | 残疾军人证、军队离退休证 |
| 残疾人 | 持残疾证（重度残疾人可带 1 名陪护免票） | 残疾证 |
| 消防救援人员 | 在职及离退休消防员 | 消防员证 |
| 警察 | 部分景区免票（如华山、庐山） | 人民警察证 |

---

## 半价人群

| 人群 | 条件 | 有效证件 |
|---|---|---|
| 学生 | 全日制大学本科及以下学历在校学生 | 学生证 + 身份证 |
| 未成年人 | 6 周岁（不含）～ 18 周岁（含） | 身份证、户口本 |
| 老年人 | 60 周岁（含）～ 65 周岁（不含） | 身份证 |
| 退役军人 | 部分景区半价 | 退役军人优待证 |
| 教师 | 无通用优惠，教师节当日部分景区凭教师证免票或半价 | 教师证 |

---

## 注意事项

1. **年龄门槛有地区差异**：大部分省份老年人免票线为 65 周岁，少数地区（如广东南澳）为 70 周岁；部分省份儿童免票线为 14 周岁以下，多数为 6 周岁以下。
2. **证件须为原件**：多数景区核验原件，电子证件不一定被认可。建议随身携带实体证件。
3. **观光车/索道另计**：景区内交通通常对免票人群半价，对 1.2m 以下儿童免票。
4. **节日特殊优惠**：建军节、教师节、警察节、重阳节等特定节日，部分景区对相关人员额外免票。
5. **需提前预约**：热门景区需通过官方平台提前预约，现场窗口核验证件。

## 临时性/特殊优惠

以下优惠具有时效性和地域性，**不保证每年都有**。AI 推荐景点时如用户身份匹配，应联网搜索「{景区名} {当前年份} 优惠政策」确认。

| 优惠类型 | 典型人群 | 说明 |
|---|---|---|
| 中/高考生 | 持准考证的应届考生 | 暑假期间（6-8 月）全国多数景区免票或半价，是最常见的暑期优惠 |
| 生肖年 | 对应生肖年份的游客 | 部分景区在特定生肖年对相应属相免票（如龙年属龙免费），城市文旅局常见活动 |
| 地区对口 | 特定省市居民 | 对口支援/帮扶城市居民凭身份证免票（如杭州对口黔东南，黔东南居民游杭州部分景区免费） |
| 特定日期 | 所有游客 | 中国旅游日（5.19）、景区周年庆、文化和自然遗产日等日期可能全员免票或半价 |
| "属"地名 | 名字含特定字的游客 | 部分景区营销活动——名字含景区地名关键字可免票（如名字含"龙"游龙门石窟） |

> 搜索关键词：`"{景区名} 2026 优惠"` `"{城市} 暑假 中高考生 免票"` `"{生肖}年 属{生肖} 免票"`

## 法律依据

- 《国家发展改革委关于进一步落实青少年门票价格优惠政策的通知》（发改价格〔2012〕283 号）
- 《退役军人保障法》——明确退役军人、"三属"人员享受优待
- 各省《景区门票及相关服务价格管理办法》

## AI 使用指导

在推荐景点时，根据用户已选择的优惠身份做具体标注：

| 用户身份 | 标注示例 |
|---|---|
| 学生 | 「持学生证半价，¥55 → ¥28」 |
| 60 岁以上老人 | 「60-65 岁半价 ¥28，65 岁以上免票」 |
| 退役军人 | 「持优待证可能免票或半价，建议出行前确认」 |
| 儿童 1.2m 以下 | 「免票」 |
| 儿童 1.2-1.4m | 「可能半价」 |
| 中/高考生 | 「持准考证可能免票，建议搜索确认」 |
| 特殊优惠线索 | 用户身份匹配时主动搜索确认（生肖/地区/特定日期） |

**不要说的话**：「有优惠」「能省不少钱」（太笼统）。**要说的话**：具体金额或比例。
